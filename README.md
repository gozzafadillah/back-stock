# Back Stock

Back Stock merupakan applikasi bertemakan management pada sebuah toko untuk mengatur produk masuk dan keluar.

### Feature :

- [x] Register
- [x] Login
- [x] CRUD Produk
- [ ] Stock

### **Yang perlu di tingkatkan**

- Alert
- Validation Form
- Loading
- Pagination
